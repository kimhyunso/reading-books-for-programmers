<h1>참조형 변수</h1>

​	참조형 변수란 메모리 번지를 참조하는 것들을 말한다. ex) String, 배열(Array), 열거형(Enum)

~~~java
String str1; //null(비어있음)
String compareStr1 = "홍길동";
String compareStr2 = "고양이";

//첫번째 비교방법
if(compareStr1 == compareStr2) //메모리 주소 같은지, false
//두번째 비교방법
if(compareStr1.equals(compareStr2)) //같은 문자열인지, false

~~~

str1은 비어있다.(null)

compareStr1과 compareStr2를 비교시, 위와 같이 첫번째 방법과 두번째 방법이 존재한다.

첫번째 방법은 서로의 메모리주소를 물어봐 false가 나온다.

두번째 방법은 서로의 문자가 같은지 확인한다.

~~~java
String str1 = new String("Test");
String str2 = new String("Test");
//두 객체는 다른 메모리주소를 참조한다.
//두 객체 비교시
if(str1 == str2) //false
if(str1.equals(str2)) //true

~~~
